Detailed Description

Calculates the total growth and maintenance respiration and returns them as 'total_growth_respiration' and 'total_maintenance_respiration', respectively.

This module is intended to be used along with the BioCro:cumulative_carbon_dynamics module, which is necessary to calculate all of its input quantities.
