Detailed Description

Uses check_leaf_temp() to calculate a temperature difference.

This module is not intended for use in crop models. Rather, it enables investigations into the leaf energy balance equations. To solve them, a value of leaf_temperature must be found such that leaf_temp_check is zero. See leaf_energy_balance() for more information.
