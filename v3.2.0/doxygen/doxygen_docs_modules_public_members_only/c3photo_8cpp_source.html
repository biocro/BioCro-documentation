49 // The variable that we call `I2` here has been described as "the useful
50 // light absorbed by photosystem II" (S. von Caemmerer (2002)) and "the
51 // maximum fraction of incident quanta that could be utilized in electron
52 // transport" (Bernacchi et al. (2003)). Here we calculate its value using
53 // Equation 3 from Bernacchi et al. (2003), except that we have replaced the
54 // factor `Q * alpha_leaf` (the product of the incident PPFD `Q` and the
55 // leaf absorptance) with the absorbed PPFD, as this is clearly the intended
56 // meaning of the `Q * alpha_leaf` factor. See also Equation 8 from the
57 // original FvCB paper, where `J` (equivalent to our `I2`) is proportional
58 // to the absorbed PPFD rather than the incident PPFD.
59 double const I2 =

62 double const J =

64 (2.0 * theta); // micromol / m^2 / s
